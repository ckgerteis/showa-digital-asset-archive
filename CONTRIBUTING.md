# Contributing

This project is a link archive and metadata register.

## What to submit

- A Fab or Sketchfab URL
- A short statement of historical fit (and what might be anachronistic)
- Any XR performance notes (scale, LOD, texture sizes, formats)
- The license type as stated on the listing (Standard or CC-BY)

## What not to submit

- Binaries copied from Fab/Sketchfab listings
- Derived redistributable packages meant to substitute for the listing
